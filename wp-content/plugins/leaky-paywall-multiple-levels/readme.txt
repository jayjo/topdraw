== Description ==

Requires zeen101's Leaky Paywall plugin.
More info at http://zeen101.com

== Installation ==

1. Upload the entire `leaky-paywall-multiple-levels` folder to your `/wp-content/plugins/` folder.
1. Go to the 'Plugins' page in the menu and activate the plugin.
